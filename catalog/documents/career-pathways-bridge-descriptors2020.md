## When to use
When explaining or designing bridge program options — how basic-skills instruction integrates with occupational content along the continuum from pre-literacy to post-secondary readiness, which EFL levels enter where, and what features each bridge model should carry.

## When NOT to use
- NOT the corpus document titled "IETP Description of Adult Education Bridge Programs" — that is a different, shorter description document; this is the full descriptor compendium. When the two disagree, flag the overlap rather than blending them (Phase A open question Q6).
- 2020-dated — the pathway framework has since moved to the 14-cluster National Career Clusters model; check the FY27 updates transcript for current framing.
