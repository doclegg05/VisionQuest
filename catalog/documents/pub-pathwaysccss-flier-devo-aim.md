## When to use
When a student is interested in apprenticeships or "earn while you learn" options — the flier covers eligibility (CTC students, pre-apprentices, veterans, dislocated workers, incumbent workers), paid on-the-job training with related technical instruction, and completion outcomes.

## When NOT to use
- Retrieval caveat: flier layout extracts fragmented text — matching leans on this note more than the document body.
- A promotional one-pager, not an application or enrollment path — route interested students to the WIOA career planner.
- AIM-WV is a USDOL grant-funded program — confirm it is still active before presenting it as available.
